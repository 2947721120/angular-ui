方法

  • href(urlMatcher, params, options)

    返回编译URL对于一个给定的URL生成方法 UrlMatcher, 填充了设置参数.

    Parameters
    参数类型细节
    urlMatcherUrlMatcher

    The UrlMatcher object which is used as the template of the URL to generate.

    参数
    (optional)
    		object(对象)

    An object of parameter values to fill the matcher's required parameters.

    选项
    (optional)
    		object(对象)

    Options object. The options are:

    • absolute - {boolean=false}, 如果属实将产生一个绝对的URL，例如 "http://www.example.com/fullurl".
    返回
    string

    Returns the fully compiled URL, or null if params fail validation against urlMatcher

    例

    $bob = $urlRouter.href(new UrlMatcher("/about/:person"), {
  person: "bob"
});
// $bob == "/about/bob";

  • sync()

    触发更新;同样的更新时地址栏URL改变这种情况，又名 $locationChangeSuccess. 当你需要使用此方法非常有用 preventDefault() 在 $locationChangeSuccess 事件， 执行一些自定义逻辑（路由保护，身份验证，配置，重定向等），然后最后进行 与通过调用过渡 $urlRouter.sync().

    Example

    angular.module('app', ['ui.router'])
  .run(function($rootScope, $urlRouter) {
    $rootScope.$on('$locationChangeSuccess', function(evt) {
      //从开始甚至停止状态变化
      evt.preventDefault();
      // 执行自定义逻辑
      var meetsRequirement = ...
      // 随着更新和状态转换，如果继续允许逻辑
      if (meetsRequirement) $urlRouter.sync();
    });
});
