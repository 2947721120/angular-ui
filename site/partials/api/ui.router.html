ui.router

对于ui.router主模块

有附带ui.router模块几个子模块，但只需要这个模块 作为您的角应用程序中的依赖。其他模块是用于组织目的.

模块是:

  • ui.router - the main "umbrella" 模
  • ui.router.router -

你需要包括 只要该模块为您的角度应用中的相关性.

<!doctype html>
<html ng-app="myApp">
<head>
  <script src="js/angular.js"></script>
  <!-- 包括： ui-router script -->
  <script src="js/angular-ui-router.min.js"></script>
  <script>
    // ...并添加 'ui.router'作为一个依赖
    var myApp = angular.module('myApp', ['ui.router']);
  </script>
</head>
<body>
</body>
</html>
